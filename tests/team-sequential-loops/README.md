# Sequential with loops Team Test

Tests sequential team execution with loops and terminate tool functionality.

## What it tests
- Sequential looping strategy with 3 agents: brainstormer, critic, coordinator
- Built-in terminate tool usage by coordinator agent
- Team conversation control and termination
- Multi-agent collaboration in iterative discussion

## Running
```bash
chainsaw test
```

The test validates that agents can coordinate in sequential looping fashion and properly terminate conversations when sufficient progress is made.
